# TaskFlow

TaskFlow is a rules-based Kanban board built with React and TypeScript.

![React](https://img.shields.io/badge/React-61DAFB?logo=react&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?logo=typescript&logoColor=white)
![Vite](https://img.shields.io/badge/Vite-646CFF?logo=vite&logoColor=white)
![CSS](https://img.shields.io/badge/CSS-1572B6?logo=css3&logoColor=white)

Unlike a traditional task board, TaskFlow uses type-safe workflow rules, priority scoring, and task constraints to create a structured task management system.

## Tech Stack

- React
- TypeScript
- Vite
- ESLint
- CSS

## Project Goals

This project focuses on demonstrating:

- Type-safe application architecture
- React component organization
- Business logic separation
- Workflow validation
- State management patterns

## Planned Features

- Kanban board
- Drag and drop
- Workflow validation
- Priority scoring
- Capacity limits
- Local storage persistence

## Getting Started

Clone the repository:

```bash
git clone https://github.com/garrib10/taskflow.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
